Face Recognition System

Face Recognition System:
programming language : Python

This is capable of detecting, identifying or verifying humans face by an image or video capture using a digital Camera. And we implemented the following features in the system:

Face recognition with name - by saving the picture with name of the person and detecting it in live feed.

Age and Gender Detection - using pretrained caffemodel(mage classification or image segmentation model).

Face Emotion Detection - using tensorflow, pretrained models.

Face Mask detection - using machine learning pretrained models of Mask and No Mask.

Face landmarks - using dlib map the facial points on a person's face.

Elder Monitoring Device

Elder Monitoring Device
This is capable of monitors the pulse rate of the elder especially when they are alone.


When the pulse rate is increase the elder ones gets altered, the device immediately sends notification of emergency alertness to the concerned guardian.


If the concerned guardian didn't respond then the information is passed to nearby healthcare providers as emergency.


The person in need , when alone can get immediate first aid and their life will be saved.
